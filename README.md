# Future-Tech

![Пример использования](./videos/FutureTechExample.gif)

**Future-Tech** — это многостраничный корпоративный сайт для компании, специализирующейся на искусственном интеллекте. Проект демонстрирует глубокое понимание нативного JavaScript, архитектуры клиентских приложений, принципов доступности (a11y) и создания полностью кастомных UI-компонентов без использования фреймворков.

Сайт рассказывает о компании, занимающейся искусственным интеллектом, и включает в себя новости, подкасты, блог и форму обратной связи.

## О проекте

Данный проект разработан как демонстрация глубокого понимания фундаментальных принципов Frontend-разработки. Основная задача — показать работу со сложной логикой на чистом JavaScript без использования готовых фреймворков. Это позволило реализовать кастомную архитектуру, включая реактивное состояние и продвинутую доступность, фокусируясь на качестве кода и производительности интерфейса.

---

## Ключевые особенности

Этот проект выделяется продуманной архитектурой и кастомными JS-компонентами, которые обычно берут готовыми из библиотек.

<br>

1. **Реактивная архитектура без фреймворков**
    - 🏗️ Все сложные компоненты наследуются от абстрактного класса BaseComponent
    - 🤖 Внутри используется JavaScript Proxy для автоматического отслеживания изменений состояния (state)
    - ⚡️ Когда данные меняются, интерфейс перерисовывается сам — без ручных вызовов render()

<br>

2. **Продвинутая система вкладок (Tabs)**
    - 🎯 Полная клавиатурная навигация (стрелки, Home, End)
    - ⌨️ Поддержка macOS (Meta + ArrowLeft/Right)
    - ♿️ Корректное управление ARIA-атрибутами
    - 🔄 Автоматическое управление фокусом и tabindex

<br>

3. **Умный кастомный селект**
    - 🔗 Полная синхронизация со скрытым нативным select
    - 📍 Авто-позиционирование: определяет свободное пространство и открывается вверх/вниз
    - 📱 На мобильных устройствах автоматически отключается в пользу нативного селекта

<br>

4. **Кастомный видеоплеер**
    - 🎮 Интуитивное управление через кастомную панель
    - 👆 Автоматическое скрытие элементов управления при воспроизведении
    - ⏯️ Отображение контролов на паузе

<br>

5. **Форма обратной связи с Firebase**
    - 📨 Асинхронная отправка данных (POST)
    - 🎯 Кастомные модальные окна на теге `<dialog>`
    - ✅ Обработка успешных и ошибочных сценариев

<br>

6. **Интерактивные элементы**
    - 📱 ExpandableContent: плавное раскрытие блоков с расчетом высоты через JS
    - 🔢 InputMask: форматирование ввода через IMask
    - 📐 Единая система rem: функция pixToRem для соблюдения пропорций дизайна

## Функционал

- **7 страниц**: Home, News, Podcasts, Resources, Blog, Contact Us и кастомная страница 404.
- **Полностью кастомные UI-компоненты**: ни одной готовой библиотеки — всё написано с нуля.
- **Адаптивная верстка**: от мобильных устройств до 4K-экранов.
- **Доступность (a11y)**: полная поддержка клавиатурной навигации и скринридеров.
- **Интеграция с Firebase**: отправка данных из формы обратной связи.

---

## Технологический стек

- **HTML5:** Семантическая верстка.
- **CSS (SASS/SCSS):** Препроцессор для более чистой и модульной стилизации. Методология БЭМ для классов.
- **JavaScript (Vanilla JS):** Чистый JavaScript без фреймворков. Использованы современные возможности (ES6+): классы, прокси, модули.
- **Firebase:** Используется как база данных для хранения сообщений из формы обратной связи.
- **IMask:** Библиотека для создания масок ввода.
- **Git:** Система контроля версий.

## Начало работы

### Установка

Следуйте этим простым шагам, чтобы запустить проект локально.

1. **Клонируйте репозиторий:**

```
   git clone https://github.com/Georgy-dev/Future-Tech
```

2. **Перейдите в директорию проекта:**

```
   cd future-tech
```

3. **Установка расширения:**
    - Откройте Visual Studio Code.
    - Перейдите в раздел Extensions (Расширения) через боковую панель или нажмите Ctrl+Shift+X (Cmd+Shift+X на macOS).
    - В строке поиска введите: Live Server.
    - Выберите расширение (автор: Ritwick Dey) и нажмите Install.

<br>

4. **Запуск проекта:**
    - Откройте папку с проектом в VS Code.
    - Выберите файл index.html в корне проекта.
      Нажмите кнопку Go Live в правом нижнем углу строки состояния (Status Bar).
    - Альтернатива: Нажмите правой кнопкой мыши на index.html в проводнике файлов и выберите Open with Live Server.

<br>

5. **Использование:**
    - Проект автоматически откроется в браузере по адресу http://127.0.0.1:5500.
    - При внесении и сохранении изменений в коде страница будет автоматически обновляться.
